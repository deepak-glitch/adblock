# SSAI (Server-Side Ad Insertion) Orchestration Endpoints
# SSAI works by stitching ads directly into the video stream on the server side.
# These are the SESSION INITIALIZATION endpoints — blocking them prevents the
# ad stitching session from being set up, so the player receives the clean stream.
# NOTE: We cannot block ad *segment* CDNs (same CDN as real content).

# ─── Google Dynamic Ad Insertion (DAI) ───────────────────────────────────────
# Used by: YouTube TV, some Hulu/Disney SSAI

dai.googlevideo.com
